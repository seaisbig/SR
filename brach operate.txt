创建仓库并指定分支名为dev
git init -b dev 
查看分支
git branch
子弹上膛
git add .
提交到版本库
git commit -m '第一次提交'
创建一个名为cs的新分支
git branch cs
修改分支的名字 cs -> abc
git branch -m cs abc
删除abc分支
git branch -d abc
若已修改未合并需要强制删除
git branch -D abc
查看分支日志
git log
分支切换
git checkout xxx
查看内容
cat a.txt
将dev分支合并到xxx(当前在xxx)
git merge dev
创建并切换
git checkout -b xyz
联系远程仓库
git remote add origin 远程仓库地址
查看已关联的远程仓库
git remote -v
确保你已经进行了本地分支的提交，并且没有冲突或错误。你可以通过执行以下命令来查看本地分支的状态：
git status
当你准备好将本地代码推送到远程仓库时，使用以下命令：
git push origin 分支名称
推送当前所在的分支
git push origin
